# Overview of the 2nd Grade Math and Writing TEKS Unit on Addition and Subtraction with Regrouping

## Unit Objectives
1. Understand place value concepts.
2. Perform addition and subtraction with and without regrouping.
3. Apply problem-solving skills in real-world contexts.
4. Develop and enhance mathematical writing skills.

## Standards Alignment
- Texas Essential Knowledge and Skills (TEKS): 2.4(A), 2.4(B), 2.4(C), 2.4(D) 
- Common Core Standards: 2.NBT.B.5, 2.NBT.B.9

## Using These Materials
This repository contains a variety of worksheets and assessments designed to support the second-grade TEKS unit on addition and subtraction with regrouping.

- **Worksheets:** These are categorized by skill level (below grade level, on grade level, enrichment) to provide differentiated instruction.
- **Assessments:** Include a quiz and a rubric for summative assessment to evaluate understanding.
- **Instructional Materials:** An outline for teaching the unit and guiding discussions is provided.
